Tripoli L2 Quiz Help

This is a quick help file to use your Android Tripoli Quiz. The Android Tripoli quiz is a very convenient way to prepare yourself to the Tripoli Level 2 where ever you are.

For best results you should use a tablet

Starting the application

Just tap the Start Quiz button, it will then randomely ask you questions.

You have 3 possibles answers and you can choose only one

Wether your answer is correct or wrong you will get a popup windows with a justification. It will sometime refer to a paragraph of the manual.

After you have answered all questions the application will let you know if you would have failed or pass the exam.

The application objective is to practice Tipoli L2 certification. The quiz contains all official Tripoli questions. This is a multilingual application so that it can be translated in other languages and help non English speaking rocketers to prepare themselves.
